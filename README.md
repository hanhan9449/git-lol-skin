# get-lol-skin

> 一个使用typescript写的lol官网壁纸爬取程序

## Get start

```shell
git clone https://github.com/hanhan9449/get-lol-skin.git
cd get-lol-skin
npm i
npm build
npm start
```

And enjoy it.

## 技术栈

1. 用typescript为程序编码
2. 用got发起http请求
3. 用winston添加日志打印功能
